Flight Planner Sweden

Kort beskrivning av webbplatsen:
- Användaren kan söka och planera flyresa mellan två svenska städer

- Användaren får sedan fram info om vald resa innehållande: 
  Från vilken stad och flygplats, till vilken stad och flygplats, valt datum,
  samt flygtid i minuter

- Användaren får också fram lite mer info om själva flyplatsen, 
  än så länge höjd över havet för från-flygplats och till-flygplats

- Användaren kan också lägga till egna svenska städer som ej finns i API:et
  för att kunna planera resor mellan dessa, då alla städer som 
  finns i Cities-api:et automatiskt läggs till i drop down-menyerna på
  planeringssidan

Webbplats-inspiration:

- Skyscanner (skyscanner.se)
- Flygresor.se (flygresor.se)
- Liknande resesök-sidor

## Bildkälla:

- Unsplash (unsplash.com)

Funktioner för framtida uppdateringar:

- Inte kunna välja samma Från och Till-stad
- Dubbelkoll för DELETE av stad
